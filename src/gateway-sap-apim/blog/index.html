About DCRP

O Domain-Centric Routing Pattern (DCRP) é um padrão arquitetural que consolida múltiplos serviços backend em um único API Proxy orientado ao domínio de negócio. Ao invés de criar proxies 1:1 para cada integração (proxy sprawl), DCRP usa roteamento metadata-driven (JavaScript + KeyValueMap) para reduzir drasticamente a complexidade operacional.
