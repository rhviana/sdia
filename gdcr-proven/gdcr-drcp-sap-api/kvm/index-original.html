Operation:

Mode 1 · HTTP Facade Canonical path → kvm.idinterface

Example: /orders/create/salesforce → dcrporderscsalesforceid01:http

Idle

Mode 2 · Manual Semantic Explicit entity / action / vendor

Compose the semantic key manually, using the same canonical mapping as the GDCR engine. The selected operation (Add / Delete) above is also applied here.

Control Plane Output Response - API Management

All operations target the KVM kvm.idinterface via the configured API facade.
